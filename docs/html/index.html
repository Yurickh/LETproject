1. Definição

O LETproject é uma iniciativa do laboratório do LET para o desenvolvimento de uma plataforma de ensino de línguas online. Seu principal objetivo é criar uma interface amigável entre alunos e professores de forma a incluir o estudo doméstico de línguas no mundo informatizado.

3. Documentação

Toda a documentação do projeto pode ser encontrada na pasta doc/.

ATENÇÃO: A documentação acima referenciada inclui explanações das funções de todas as classes e métodos implementados e tem por público alvo principalmente desenvolvedores interessados em contribuir com o projeto.

Para uma versão específica para usuários, ou seja, pessoas que estão interessadas no funcionamento da ferramenta e não em seus mecanismos, leia o ponto 4. Instalação e execução

3.i. Doxyfile (html)

No intuito de simplificar a navegação dentro da documentação do projeto, utilizamos a ferramenta Doxygen, gerando, assim, um arquivo HTML que contém uma interface amigável. Este arquivo está contido em doc/html/ e para acessá-lo basta abrir o arquivo doc/html/index.html com o seu navegador.
